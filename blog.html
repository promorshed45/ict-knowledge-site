var, let, and const

আমরা var দিয়ে কোন ভ্যারিয়েবল declare করলে সেটা পুনরায় re-declare এবং re-assign করা যাবে । তবে let দিয়ে ভ্যারিয়েবল declare করলে সেইটা re-assign করা যাবে কিন্ত re-declare করা যাবে না এবং const দিয়ে ভ্যারিয়েবল declare করলে সেইটা re-declare এবং re-assign কোনটি করা যাবে না।

null and undefined

null:

জাভাস্ক্রীপ এ null এবং undefined এই দুটি datatype Operator এর মধ্যে পড়ে। null বলতে আসলে জাভাস্ক্রীপে কোন কিছুর অনুপস্থিতি বোঝায়। জাভাস্ক্রিপ এ নালকে একটি অবজেক্ট হিসেবে বিবেচনা করা হয়। জাভাস্ক্রীপে একটি variable কে আনডিফাইন্ড হিসেবে ডিক্লেয়ার করেও খালি একটি অবজেক্ট তৈরী করা যায়।

undefined

জাভাস্ক্রিপ্টে undefined হলো একটি variable যার কোনো value থাকে না। একটি ভেরিযবলকে আনডিফাইন্ড ডিক্লেয়ার করার মাধ্যমেও ভেরিযবলের ভেলু খালি করা যায়।

আনডিফাইন্ড এবং নালের মধ্যে পার্থক্য:
typeof undefined // undefined
typeof null // object
null === undefined // false
null == undefined // true

REST API

API এর পূর্ণরূপ:

API - Application Programming Interface

API ব্যবহার করে আমরা অন্য কোন সাইট হতে আমাদের সাইটে বিভিন্ন তথ্য নিয়ে আসতে পারি।

Rest API এর পূর্ণরূপ:

Rest API - Representational State Transfer Application Programming Interface

Rest API প্রত্যেকটি URL কে কিছু অবজেক্টকে রিপ্রেজেন্ট করে । রেস্ট এর সবচেয়ে ছোট সংঙ্গা এর নামেই দেয়া আছে, “Representational State Transfer” অর্থাৎ এটি কোন কিছুর অবস্থা বা স্টেট একটি রিপ্রেজেন্টেশন এর মাধ্যমে ক্লায়েন্ট টু সার্ভার এবং সার্ভার টু ক্লায়েন্টে ট্রান্সফার করে । রেস্ট এপিআই ডেভেলপমেন্ট এর কোন স্ট্যান্ডার্ড নয়, এটি নেটওয়ার্ক এ্যাপ্লিকেশন ডিজাইনের জন্য আর্কিটেকচারাল স্টাইল ।
